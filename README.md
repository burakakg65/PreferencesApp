# Preferences & Authentication App

Bu proje, React Native ve Expo kullanılarak geliştirilmiş basit bir **kimlik doğrulama ve tercih yönetimi** uygulamasıdır.  
Uygulamada kullanıcı giriş yapabilir ve giriş durumuna göre ekranlar arasında yönlendirilir.

---

##  Kullanılan Teknolojiler

- React Native
- Expo
- React Navigation (Native Stack)
- Context API
- AsyncStorage

---

##  Uygulama Özellikleri

- Kullanıcı giriş ekranı (Login Screen)
- Context API ile global kullanıcı durumu yönetimi
- AsyncStorage ile oturum bilgisinin saklanması
- Stack Navigator ile sayfa geçişleri
- Basit ve anlaşılır ekr
---
